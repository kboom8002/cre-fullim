# 🏢 공공데이터 API & 웹 크롤링 연동 감사 및 초심자 가이드

이 가이드는 **cre-fullim** (Full IM Studio) 및 **cre-dealcard** (MVP DealCard) 프로젝트 시스템에 구축된 공공데이터 연동 및 웹 크롤링 내역을 감사해 정리하고, 개발 지식이 적은 비전공자나 초심자도 각 외부 공공데이터 API 승인 및 인증키 발급을 혼자 힘으로 마칠 수 있도록 단계별로 가이드합니다.

---

## 🔍 Part 1. 외부 연동 및 크롤링 내역 감사 요약

우리 시스템은 부동산 브로커의 비정형 메모와 수동 입력 한계를 보완하기 위해 **외부 공공 API 데이터를 통한 실시간 보강 설계**를 완전히 이식했습니다.

### 1.1 공공데이터 및 로컬 API 연동 감사

| 카테고리 | 파일 경로 | 연동 목적 및 취득 데이터 | 환경변수명 | 난이도 |
| :--- | :--- | :--- | :--- | :--- |
| **주소 정규화** | `address-resolver.ts` | 자유형식 주소 입력 시 도로명/지번 정규화, 법정동코드(10자리), PNU코드(19자리), 시군구코드(5자리) 추출 | `JUSO_CONFIRM_KEY` | 🟢 하 (즉시승인) |
| **건물 제원** | `building-register-api.ts` | 정부 24 건축물대장 API 연동. 연면적, 대지면적, 용도, 주구조, 층수, 승인일, 건폐율/용적률 취득 | `DATA_GO_KR_API_KEY` | 🟢 하 (즉시승인) |
| **공시지가** | `land-price-api.ts` | 국토부 개별공시지가 API 연동. 필지별 ㎡당 공시지가 취득 후 평당가 환산 | `DATA_GO_KR_API_KEY` | 🟢 하 (즉시승인) |
| **공법/용도** | `land-use-api.ts` | LURIS 토지이용계획 API 연동. 용도지역(상업/주거 등), 기타 용도지구, 법정 건폐율/용적률 상한 | `DATA_GO_KR_API_KEY` | 🟢 하 (즉시승인) |
| **매각 거래** | `real-transaction-api.ts` | 국토교통부 실거래 정보 API 연동. 시군구 내 상업용/근생 거래 사례 수집 및 평균 시세 도출 | `DATA_GO_KR_API_KEY` | 🟢 하 (즉시승인) |
| **지도 & POI** | `kakao-map-api.ts` | 카카오 로컬 REST API 연동. 인근 지하철역 정보 검색, 반경 500m 이내 생활 편의시설(카페, 주차 등) 밀집도 계산 | `KAKAO_REST_API_KEY` | 🟢 하 (즉시승인) |

### 1.2 웹 크롤링 및 수동 리포트 연동 감사

| 수집 대상 | 크롤링 방식 및 URL | 타겟 데이터 | 활용처 |
| :--- | :--- | :--- | :--- |
| **글로벌 시장 분기 리포트** | JLL, CBRE, Savills Korea 공식 홈페이지 PDF | 프라임 오피스 권역별(GBD, CBD, YBD) 공실률 및 시장 평균 Cap Rate | `micro_market_demand_story` (시장 수요 분석) |
| **상권 매출 정보** | 소상공인시장진흥공단 상권분석 시스템 | 행정동 기준 카드 매출 추이 및 요식/리테일 개폐업 지수 | `micro_market_demand_story` (상권 매력도) |
| **민간 호가 분석** | 네이버 부동산 매물 정보 | 주변 경쟁 오피스/근생 매물의 실시간 평당 호가 지표 | `valuation_logic_comparables` (비교 사례 분석) |

> [!TIP]
> **웹 크롤링의 안정성 극대화 방안**
> 민간 시세 정보 플랫폼은 지속적으로 HTML 마크업을 변경하고 크롤러 IP를 차단하므로, 직접적인 스크래핑보다는 공용 Open API(소상공인시장진흥공단) 연동 및 분기별 글로벌 컨설팅사 PDF 시장 리포트를 파싱하는 방식을 권장합니다.

## 🛠️ Part 2. 초심자용 공공데이터 API 발급 해설 가이드

---

### 1️⃣ 도로명주소 API 인증키 발급 (행정안전부)
* **목적**: 브로커가 "역삼동 823-21" 이라고 대충 적어도 "서울특별시 강남구 테헤란로 123"과 같이 깨끗한 주소로 보정하고, 모든 공공 API의 Key가 되는 **19자리 PNU 코드**를 구성해 줍니다.
* **환경변수명**: `JUSO_CONFIRM_KEY`

#### [발급 절차]
1. 인터넷 브라우저를 열고 **[도로명주소 개발자센터](https://business.juso.go.kr/)**에 접속합니다.
2. 상단 메뉴의 **[API 신청] -> [도로명주소 API 신청]**을 클릭합니다.
3. 로그인 화면이 나오면 회원가입 후 로그인합니다. (비회원 신청도 가능하나, 관리와 재발급을 위해 가입을 강력 권장합니다.)
4. **신청서 작성 페이지**에서 정보를 아래와 같이 입력합니다:
   * **신청 구분**: `Open API (웹/앱 서비스용)`
   * **사용 용도**: `시스템 개발 및 주소 보정용`
   * **망 구분**: `인터넷망`
   * **서비스 종류**: `도로명주소 검색 API` (또는 `좌표 제공 API`)
   * **URL 정보**: 개발 단계에서는 `http://localhost:3005` (실제 개발 환경 포트) 또는 `http://localhost:3000`을 적어주시면 됩니다.
5. **[신청하기]** 버튼을 클릭하면 **즉시 승인**과 함께 영문+숫자로 구성된 긴 **인증키(Confirm Key)**가 발급됩니다.
6. 발급된 키를 복사하여 `.env.local`의 `JUSO_CONFIRM_KEY=` 옆에 붙여넣습니다.

### 2️⃣ 공공데이터포털 API 인증키 발급 (data.go.kr)
* **목적**: 건축물대장, 개별공시지가, 토지이용계획, 실거래가 API를 한곳에서 일괄 신청할 수 있는 **대한민국 정부 공식 포털**입니다.
* **환경변수명**: `DATA_GO_KR_API_KEY` (하나의 포털 키로 4개 API에 공용 사용)

#### [가장 중요한 선행 작업: 공공데이터포털 회원가입]
1. **[공공데이터포털 공식 홈페이지](https://www.data.go.kr/)**에 접속하여 우측 상단의 **[회원가입]**을 진행합니다.
2. 로그인을 완료합니다.

#### [세부 API별 검색 및 신청 방법]
공공데이터포털 검색창에 아래의 서비스 이름을 하나씩 검색하여 **[활용신청]**을 클릭해야 합니다.

#### ① 국토교통부_건축물대장정보 서비스 (건물 제원용)
* **검색어**: `국토교통부 건축물대장정보`
* **신청하기**: 검색 결과에서 **[국토교통부_건축물대장정보 서비스]** 오픈 API 항목을 클릭한 뒤, 우측의 **[활용신청]** 버튼을 누릅니다.
* **활용 목적 기재**: `상업용 부동산 데이터 시스템 개발 및 예비 투자 설명서 자동 생성`
* **시스템 유형**: `1인 개발/기타` 선택 후 **[신청]** 클릭. (자동 즉시 승인됩니다.)

#### ② 국토교통부_개별공시지가정보 서비스 (공시지가용)
* **검색어**: `국토교통부 개별공시지가`
* **신청하기**: 동일하게 오픈 API 서비스를 찾아 **[활용신청]**을 클릭하고 즉시 승인을 받습니다.

#### ③ 국토교통부_토지이용계획정보 서비스 (용도지역/지구 제한사항 확인용)
* **검색어**: `국토교통부 토지이용계획정보`
* **신청하기**: 동일하게 **[활용신청]**을 클릭하여 승인을 획득합니다.

#### ④ 국토교통부_상업업무용 부동산 실거래가 자료 (주변 매각 비교 사례용)
* **검색어**: `상업업무용 부동산 매매 실거래가 자료`
* **신청하기**: 동일하게 **[활용신청]**을 완료합니다.

#### [인증키 확인 방법]
1. 공공데이터포털 우측 상단의 **[마이페이지] -> [개발계정]**으로 이동합니다.
2. 방금 신청한 API 목록이 보이며, 목록 중 하나를 클릭하면 상세 페이지에 **[일반 인증키(Encoding)]**와 **[일반 인증키(Decoding)]** 두 종류가 표시됩니다.
3. 우리 Next.js 프로젝트에서는 특수문자가 변형되지 않도록 안전하게 **[일반 인증키(Decoding)]**를 복사합니다.
4. `.env.local` 파일에 `DATA_GO_KR_API_KEY=복사한디코딩키` 양식으로 적용합니다.

### 3️⃣ 카카오 API 인증키 발급 (developers.kakao.com)
* **목적**: 지도 렌더링(Kakao Map SDK)과 서버사이드 역세권/주변 인프라(POI) 고해상도 검색을 지원합니다.
* **환경변수명**:
  * `NEXT_PUBLIC_KAKAO_MAP_KEY` (지도 렌더링용 앱 키)
  * `KAKAO_REST_API_KEY` (서버 통신용 REST API 키)

#### [발급 절차]
1. **[카카오 디벨로퍼스 홈페이지](https://developers.kakao.com/)**에 접속하여 로그인(카카오 계정)합니다.
2. 우측 상단의 **[내 애플리케이션] -> [애플리케이션 추가하기]**를 선택합니다.
   * **앱 이름**: `CRE Mobile IM Studio`
   * **사업자명**: `개인 개발` (아무 텍스트나 무방)
   * 저장 버튼을 누릅니다.
3. 생성된 애플리케이션을 클릭하면 대시보드 화면에 **네 종류의 키(Key)**가 발급되어 있습니다:
   * **REST API 키**: 서버사이드 POI 검색에 사용되므로, 복사하여 `.env.local`의 `KAKAO_REST_API_KEY=` 옆에 저장합니다.
   * **JavaScript 키**: 웹 지도 로드에 필요하므로, 복사하여 `NEXT_PUBLIC_KAKAO_MAP_KEY=` 옆에 저장합니다.

#### ⚠️ [절대 빼먹으면 안 되는 필수 설정: 플랫폼 등록]
카카오는 보안을 위해 등록된 주소(도메인)에서만 지도를 보여줍니다.
1. 왼쪽 메뉴 목록에서 **[플랫폼]**을 선택합니다.
2. **[Web 플랫폼 등록]** 버튼을 클릭합니다.
3. **사이트 도메인**에 아래의 개발 및 프로덕션 주소를 엔터로 구분하여 모두 입력합니다:
   * `http://localhost:3005` (로컬 cre-fullim)
   * `http://localhost:3000` (로컬 cre-dealcard)
   * `https://cre-fullim.vercel.app` (추후 배포될 배포 도메인)
4. 저장 버튼을 누릅니다. (이제 로컬 화면에서도 지도가 정상 작동합니다.)

## 📝 Part 3. 프로젝트 `.env.local` 세팅 템플릿

위 단계들을 통해 발급받은 실제 인증키들을 아래 템플릿 파일에 그대로 복사-붙여넣기하여 `cre-fullim` 폴더와 `cre-dealcard` 폴더 루트의 `.env.local`에 배치해 주시면 연동이 활성화됩니다.

```env
# ==========================================
# 🏢 CRE REAL ESTATE API CONFIGURATION
# ==========================================

# 1. 공공데이터포털 (data.go.kr) - 디코딩(Decoding) 키 권장
DATA_GO_KR_API_KEY=your_decoded_api_key_here

# 2. 행정안전부 도로명주소 API 인증키 (juso.go.kr)
JUSO_CONFIRM_KEY=your_juso_confirm_key_here

# 3. 카카오 지도 API 앱 키 (developers.kakao.com)
# JavaScript 키 대입 (프론트엔드 HTML 지도 로드용)
NEXT_PUBLIC_KAKAO_MAP_KEY=your_kakao_javascript_key_here

# REST API 키 대입 (백엔드 지하철역 및 POI 반경 검색용)
KAKAO_REST_API_KEY=your_kakao_rest_api_key_here

# 4. 브이월드 오픈 API (vworld.kr) - 용도지역 지적도 시각화 확장용
VWORLD_API_KEY=your_vworld_api_key_here
```

> [!NOTE]
> **API 키 미세팅 상태에서의 안전 장치**
> 우리 시스템은 초심자가 API 키를 아직 받지 못한 오프라인 상태에서도, 입력된 건물 주소를 지능적으로 분석하여 역삼 권역에 맞는 모킹 데이터를 반환해 줌으로써 화면이나 로직이 크래시되지 않고 데모를 유려하게 이어갈 수 있도록 설계되었습니다.
