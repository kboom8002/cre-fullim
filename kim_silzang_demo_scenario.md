# 김 실장 데모 시나리오 — "오늘 하루가 달라집니다"

> **데모 목표**: 60대 꼬마빌딩 브로커가 '나도 이거 쓸 수 있겠다'고 느끼는 순간을 5번 만든다.
> **총 소요 시간**: 약 20분
> **배경**: 성수권역 80억대 근생형 꼬마빌딩 1건을 하루 동안 처리하는 흐름

---

## 🎬 오프닝 멘트 (1분)

**[진행자]**

> "오늘 데모는 복잡한 기능 설명이 아닙니다.
> 중개업 22년차 김 실장이 오늘 아침부터 저녁까지
> 어떻게 일이 달라지는지를 함께 보시는 겁니다.
> 카톡 하나가 거래로 이어지는 과정입니다."

## Scene 1. 아침 7시 — 카톡 매물 → 1분 딜카드

### 상황

> 김 실장, 아침에 카톡 단톡방에서 매물 정보 수신.
> "성수동 근생 80억대, 1층 F&B 가능, 사옥수요도 볼 수 있음. 주소는 비공개."

### 데모 실행 (화면)

1. 딜카드 홈 접속 → **"카톡 매물 설명을 붙여넣으세요"** 텍스트박스 등장
2. 카톡 메시지 그대로 붙여넣기 (김 실장이 매일 하는 동작)
3. **"1분 딜카드 만들기"** 탭

```
생성 중 화면:
✅ 권역·자산유형 파악 중
✅ 딜 포인트 정리 중
✅ 주소·민감정보 숨기는 중
```

**결과 화면 등장 (30초)**:

```
🏢 건물 신호 요약
  권역: 성수·뚝섬권
  자산유형: 근생형 꼬마빌딩
  가격대: 80억대
  현재상태: 일부 임대 중

🔒 자동으로 숨긴 정보
  • 정확한 주소
  • 임차인명
  • 호실별 임대료

💬 카톡 문구 (복사 버튼)
  "성수권역 80억대 근생형 자산입니다.
   사옥+부분임대형 매수자에게 검토 가치가 있고..."
```

### ⚡ 첫 번째 Jaw Drop

**[진행자]**

> **"30초 만에 카톡으로 보낼 딜카드가 나왔습니다.
> 주소도 안 알려주고, 임차인명도 안 나오고.
> 김 실장님, 예전에 이 정리를 어떻게 하셨어요?"**

## Scene 2. 오전 9시 — 매수자 조건 → Buyer Fit Memo

### 상황

> 오래된 고객 박 대표가 카톡으로 조건을 보내옴.
> "50~80억, 성수 or 강남, 사옥 겸 임대 원함. 주차 중요."

### 데모 실행

1. 브로커 홈 → **"매수자 조건 정리하기"**
2. 박 대표 카톡 붙여넣기
3. 결과:

```
📋 박 대표 매수 조건 요약
  예산: 50억~80억
  지역: 성수, 강남
  목적: 사옥 + 일부 임대수익
  필수: 주차, 사옥 전환 가능
  리스크 성향: 중간

💬 매수자에게 보낼 문구:
  "대표님, 성수권역 80억대 근생형 물건이
   말씀하신 조건과 잘 맞습니다..."
```

4. **"딜카드와 연결"** → 방금 만든 딜카드와 매칭 완료

### ⚡ 두 번째 Jaw Drop

**[진행자]**

> **"방금 카톡에서 복사한 조건으로 Buyer Memo를 만들고,
> 30분 전에 만든 딜카드와 자동으로 연결했습니다.
> 두 개 카톡 메시지가 딜이 됐습니다."**

## Scene 3. 오전 10시 — 현장 방문 → AiPage 임대 마케팅

### 상황

> 건물주 이 사장이 전화.
> "1층이 6개월째 공실이에요. 어떻게 해볼 수 없나요?"
> → 김 실장, 현장 방문 + 사진 30장 촬영

### 데모 실행

1. AiPage → **"새 공간 등록"**
2. **사진 30장 일괄 업로드**
3. AI 자동 분류:

```
✅ 외관 사진 4장 → 입지·접근성 앨범
✅ 내부 사진 12장 → 실내 환경 앨범
✅ 주차 사진 3장 → 편의시설 앨범
⚠️ 야간 사진 없음 (야간 업종 테스트 권장)
```

4. **"AI 임대 페이지 생성"** → 30초 뒤 완성
5. 카카오 홍보 카피 자동 생성:

```
"성수권역 사옥 겸 임대 가능 빌딩
 도보 5분, F&B 적합, 주차 가능
 ─ 링크로 공간 전체 확인하기 →"
```

6. **건물주 보고서 생성**:

```
📊 이 사장님 이번 주 임대 마케팅 현황
  • 카카오 채널 노출: 1,240회
  • 링크 클릭: 87회
  • 상세 문의: 3건
  • 관심 업종: F&B 2건, 사무실 1건
```

### ⚡ 세 번째 Jaw Drop

**[진행자]**

> **"사진 찍고 업로드만 했는데,
> 건물주한테 '이번 주 1,240명한테 노출됐다'는
> 보고서가 생겼습니다.
> 전속 계약이 왜 유지되는지 아시겠죠?"**

## Scene 4. 오후 2시 — 박 대표 상세 자료 요청 → Mobile IM

### 상황

> 오전에 딜카드를 보낸 박 대표가 카톡.
> "김 실장님, 조금 더 자세한 자료 있으세요?"

### 데모 실행

1. 딜카드 결과 화면 → **"📱 Mobile IM 만들기 (3분)"** 버튼 탭

```
생성 중 (3분 소요):
✅ 물건 개요 정리
✅ 입지·상권 분석
✅ 임대 현황 요약
✅ 수익 분석 (추정치)
✅ 확인 필요 사항
✅ 투자 포인트
✅ 다음 단계 안내
```

2. 모바일 웹 링크 생성: `https://js.im/m/abc123`
3. 카톡 공유 텍스트:

```
📋 성수권역 80억대 근생형 자산
   투자 메모 (Mobile IM)

✅ 수익률(추정): 약 4.8%
✅ 임대율: 85%
⚠️ 확인 필요: 임대차 만기, 주차 조건

👉 상세 보기: https://js.im/m/abc123

※ 본 자료는 예비 검토용이며 투자 권유가 아닙니다.
```

4. **박 대표 핸드폰으로 링크 접속** → 7섹션 스와이프 카드 뷰어 시연

### Mobile IM 뷰어 시연 (화면 확대)

```
← 스와이프 →  3/7  📊 수익 분석

• 현재 임대율 85% 기준 추정 수익률 3.5%
• 1층 리테일 임대 완료 시 4.8% 타겟
• 주의: 정확한 수치는 임대차계약서 확인 후 산정

━━━━━━━━━━━━━━━━━━━━━━
⚖️ 본 자료는 예비 검토용입니다.

📊 Full IM 준비도: 52/100
   □ 임대차 요약표  □ 등기부등본
   [상세 IM 요청하기 →]
```

### ⚡ 네 번째 Jaw Drop

**[진행자]**

> **"딜카드 하나로 시작해서,
> 3분 만에 7페이지짜리 투자 설명서가 모바일로 만들어졌습니다.
> 예전에 IM 외주 맡기면 얼마였죠?
> 2주에 500만원? 이게 3분입니다."**

## Scene 5. 오후 4시 — 박 대표 진지해짐 → Full IM 전환

### 상황

> 박 대표가 현장 보고 싶다고 연락.
> "자료 좀 더 자세하게 볼 수 있을까요? 세무사한테도 보여줘야 해서요."

### 데모 실행

1. Mobile IM 하단 → **"상세 IM 요청하기"** 탭
2. Full IM 준비도 화면:

```
📊 Full IM 준비도: 52/100

현재 가능한 것:
✅ 블라인드 티저
✅ Mobile IM (7섹션)
⚠️ Full IM은 추가자료 필요

Full IM 완성을 위한 2가지:
□ 임대차 요약표 (+15점)
□ 등기부등본 (+15점)

추가하면 준비도 82점 → 전문가 검토 후 Full IM 발행 가능
```

3. 건물주 이 사장에게 자료 요청 → 이틀 뒤 서류 수령
4. Full IM Studio → **18섹션 자동 생성** + 세무사 섹션별 배정
5. 전문가 패치 완료 → **"Buyer-ready 승인"** → PDF 발행

### 전문가 배정 화면

```
📎 섹션별 전문가 검토 현황
  ✅ 임대수입·NOI 분석   →  CRE 컨설턴트 (완료)
  ✅ 가치평가 비교사례   →  감정평가사 (완료)
  ✅ 토지·법적 제약      →  부동산 전문 변호사 (완료)
  ⏳ 대출 민감도 분석   →  금융 전문가 (검토 중)
```

### ⚡ 다섯 번째 Jaw Drop

**[진행자]**

> **"카톡 메시지에서 시작해서,
> 딜카드 → Mobile IM → 전문가 검토 Full IM까지
> 김 실장 혼자 만들었습니다.
> 세무사, 감정평가사는 자기 섹션만 검토하면 됩니다.
> 이게 거래 한 건의 전체 흐름입니다."**

| 예상 질문 | 답변 |
|---|---|
| "AI가 쓴 걸 믿을 수 있나요?" | "모든 수치에 '추정', '확인 필요' 레이블이 붙습니다. 과장 표현은 자동으로 차단됩니다. 세무사/감정평가사가 핵심 섹션을 검토합니다." |
